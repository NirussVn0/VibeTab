# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-29
### Initial Release
*   **Core:** Grid-based layout engine with auto-collision and database persistence.
*   **UI:** Modern Glassmorphism design with Teal/Cyan theme.
*   **Features:**
    *   Drag-and-Drop widget management.
    *   Settings Panel with Import/Export.
    *   Command Palette (`Ctrl+K`).
    *   Background Engine (Image/Video/GIF support).
*   **Widgets:**
    *   Clock (Digital/24h).
    *   Search (Google/Bing/DDG).
*   **Performance:**
    *   IndexedDB storage for media.
    *   Lazy-loaded components.
    *   Offline-first architecture.
